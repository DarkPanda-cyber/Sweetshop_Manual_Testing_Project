# 🍬Sweetshop Manual Testing Project

A complete manual testing project performed on the Sweetshop demo e-commerce application.

This project demonstrates the complete Software Testing Life Cycle (STLC) including planning, test design, execution, responsive testing, defect reporting, and documentation.

---

## Project Overview

**Application:** Sweetshop Demo Website

**Application Type:** E-commerce Web Application

**Testing Type:** Manual Testing

**Testing Approach:** Black Box Testing

**Testing Duration:** Self Project

---

## Objectives

The objective of this project was to validate the core functionality of the Sweetshop web application by testing:

- User Authentication
- Product Display
- Navigation
- Shopping Basket
- Checkout Flow
- Account Dashboard
- Responsive Behaviour
- UI Validation

## Major Bugs Identified

- User login accepts invalid credentials
- Checkout accepts invalid payment details
- Shipping calculation returns **£NaN**
- Shipping calculation displays **£61.99 instead of £7.99**
- Checkout fails on mobile devices
- Broken About page navigation
- Broken product image
- Broken social media links
- Dashboard favourite product calculation incorrect
- Previous orders sorting behaves unexpectedly
